Evaluation

YOLO or known as You Only Look Once is an algorithm made to detect patterns using computer vision. This software has been in ongoing development since 2015. Making this software relatively new. Comparing to other Real-time detection models YOLO uses a different method of detection by calculating the object class and bounding boxes all at once. Whereas other models will follow the usage of Identifying the objects then applying bounding boxes based on the overall probability. Making YOLO capable of operating much faster in Realtime environments with great accuracy.

While experimenting with YOLO I found that while the software functions as expected. Quite a few limitations as well. YOLO can accurately detect objects (Average of 81.9%) in its field of view but suffers recognising objects obfuscated by shadows or at great distances. Outliers noticed were detections made on advertisements or reflections the program struggled with distinction in these situations.

I would recommend this software to AT Transport as its able to recognise objects at a reasonable field of view and accuracy. This would work well in short distance based locations such as intersections and general traffic light stops.

As this tool is still being developed it would be recommended to AT Transport to further research and expand the data set to rule out possible outliers which will arise. Example Unknown transportation types such as electric scooters which are not currently detected by this software.
