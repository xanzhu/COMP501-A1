Evaluation

YOLO or known as You Only Look Once is an algorithm made to detect patterns using computer vision. This software has been in ongoing development since 2015. Making this software relatively new. Comparing to other Real-time detection models YOLO uses a different method of detection by calculating the object class and bounding boxes all at once. Whereas other models will follow the usage of Identifying the objects then applying bounding boxes based on the overall probability. Making YOLO capable of operating much faster in Realtime environments with great accuracy.

While experimenting with YOLO I found that while the software functions as expected there is a lot of limitations. YOLO can accurately detect objects in its field of view but suffers recognising objects obfuscated by shadows or at great distances. There also was the issue where detections were made on advertisements or reflections the program struggled with distinction in these situations.

I would recommend this software to AT Transport as its able to recognise objects at a reasonable field of view. This would be helpful in identifying in locations such as intersections but may struggle with object detection at great distances.

As this tool is still being developed it would be recommended to AT to further research and expand the data set to rule out possible outliers which will arise from unknown transportation types such as electric scooters which are not currently detected by this software.
